# Summary

Nintendo Kirby Whispy Woods Inspired Plant Pot.

This plant pot is inspired by Whispy Woods, the tree boss from Kirby. 

This garden accessory is printed in Polywood and helped bring the character to life, it's up to you to give them a bushy head of leaves that it so desperately needs! 

You may need to scale him up a bit in which ever 3D print software you use is, as I made the design smaller than I intended to.
